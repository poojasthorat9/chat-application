# chat-application
Client-Socket Chat Application using Nodejs
# Requirement to run this program
1. Need a editor to write your source code e.g. atom
2. You need npm & socket.io packages which you can download from https://nodejs.org/en/download/
   you need to download binary package according to your operating system.
3. Basic steps for Installation of package:http://blog.teamtreehouse.com/install-node-js-npm-linux
   This for linux you can choose your operating system for this and follow this basic steps to run your program.
4.What you need to do is run a command in the Shell of that linux machine:
   node index.js and your program will show output in browser @localhost:3000
